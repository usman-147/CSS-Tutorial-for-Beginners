Relative units are different from pixels, they vary from website to website, or any other tags within the website.

        1) % : if parent div has size 100% and child div 2 has size 33%, this indicates that it's taking 33% from it's parent div, which now contains 77%
        2) em : font-size depends on parent, width depends on it's text itself
        3) rem : font-size depends on the root element such as html or body
        4) vh : viewport/browser height: takes reference from browser
        5) vw : viewport/browser width: takes reference from browser
